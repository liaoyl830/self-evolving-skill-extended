## 1️⃣ sessions_yield 机制分析

### 1.1 是什么

`sessions_yield` 是 OpenClaw v2026.3.12 新增的子代理协调功能：

- **作用:** 让主代理立即结束当前回合
- **效果:** 跳过排队工具工作，携带隐藏跟进负载到下一回合
- **适用场景:** 多代理编排、长时间任务协调

### 1.2 工作原理

```
传统流程:
主代理 → 子代理任务 → 等待完成 → 收集结果 → 回复用户
        (阻塞)         (可能数分钟)

sessions_yield 流程:
主代理 → 子代理任务 → yield 立即返回 → 后台继续 → 完成后通知
        (不阻塞)      (秒级响应)      (异步)
```

### 1.3 API 形式

```typescript
// 在子代理任务中使用
await sessions_yield({
  message: "后台处理中，完成后通知您",
  // 隐藏的跟进负载，用户看不到
  followUp: {
    task: "continue_processing",
    sessionId: "xxx"
  }
});
```

### 3.2 实施步骤

#### 步骤 1: 修改复盘触发逻辑

```python
# 当前 (阻塞)
def daily_review():
    result = engine.execute(...)  # 阻塞
    report = generate_report()    # 阻塞
    write_to_memory(report)       # 阻塞
    return f"✅ 复盘完成：{report}"

# 优化后 (异步)
def daily_review():
    # 立即返回
    sessions_yield(
        message="🫡 收到，开始复盘今日任务...",
        followUp={
            "task": "daily_review_background",
            "timestamp": datetime.now().isoformat()
        }
    )
    
    # 后台继续
    background_task = spawn_subagent(
        task="daily_review_background",
        on_complete="notify_user_with_report"
    )
    
    return None  # 不阻塞主会话
```

#### 步骤 2: 创建后台复盘代理

```python
# review_background_agent.py
class ReviewBackgroundAgent:
    def __init__(self, engine):
        self.engine = engine
    
    async def run(self, context):
        # 执行技能
        result = self.engine.execute(
            skill_id=context['skill_id'],
            context=context,
            success=True,
            value=0.95
        )
        
        # 生成报告
        report = self.generate_report(result)
        
        # 写入文件
        await self.write_to_memory(report)
        
        # 完成通知
        await self.notify_completion(report)
```

#### 步骤 3: 完成通知机制

```python
async def notify_completion(self, report):
    """复盘完成后通知用户"""
    
    # 方式 1: 系统事件
    await openclaw_system_event(
        text=f"✅ 今日复盘完成！\n\n{report.summary}",
        mode="now"  # 立即唤醒
    )
    
    # 方式 2: 消息发送
    await openclaw_message_send(
        target="current_session",
        message=f"📊 复盘报告已生成：\n\n{report.full_text}",
        attachments=[report.file_path]
    )
```
